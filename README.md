# \# DA-1: iGaming KPI Analytics Suite

# 

# End-to-end analytics project for a fictional iGaming operator (GreenSpin Gaming), covering GGR, NGR, churn, LTV, and bonus efficiency across 50,000 simulated players.

# 

# \## Live Dashboard

# \[View on Tableau Public](https://public.tableau.com/app/profile/oluwatobi.bamidele/viz/GreenSpinGamingiGamingKPIDashboard/iGamingKPIDashboard)

# 

# \## Project Overview

# This project simulates the full analytics workflow of a real iGaming data analyst:

# \- Generated a realistic dataset of 50,000 players and 2.1 million gaming sessions using Python

# \- Loaded data into PostgreSQL and wrote 10 analytical SQL queries

# \- Built a 4-view Tableau dashboard covering GGR trend, LTV by segment, churn heatmap, and bonus vs revenue

# \- Produced a 1-page CMO executive summary with 5 data-backed business recommendations

# 

# \## Key Findings

# \- VIP players (5% of base) generate 49% of total GGR

# \- Top 1,000 players (2%) account for 33.83% of all revenue

# \- Dormant players churn at 93.66% within 83 days of registration

# \- Free Spins delivers the highest bonus ROI at 106x

# \- Malta and Canada produce the highest average LTV per player at EUR 2,756 and EUR 2,700

# \## How to Reproduce

# 1\. Run `generate\_igaming\_data.py` to generate the three CSV files

# 2\. Load CSVs into PostgreSQL using the COPY commands in the project notes

# 3\. Run the 10 SQL queries against your local database

# 4\. Open Tableau Public and connect to the exported CSV files
